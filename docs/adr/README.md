# Architecture Decision Records

This directory holds the project's ADRs — short, dated notes pinning a
decision and the alternatives considered. Format follows the
lightweight Michael Nygard shape: Status, Context, Decision,
Consequences, Alternatives. One file per decision, numbered
monotonically.

Numbering is `NNNN-kebab-case-title.md` starting at `0001`. ADRs are
append-only: a later ADR supersedes an earlier one rather than editing
history.

- [0001](0001-action-item-deletion-semantics.md) — Action-item deletion semantics in the IPC write-back path. Accepted 2026-05-01.
